git branch  查看分支 
* master前的*表示当前处于该分支
git branch a  创建一个a新分支
git checkout a 切换到a分支
git checkout -b a 创建分支a并切换到分支a
git merge a 将分支a合并到当前分支，如果有冲突时不能合并，需要先解决冲突才能合并。
git branch -d a 删除分支a，合并到主分支后才可以删除
git branch -D a 强制删除分支a，不需要提前合并
git tag v1.0
